                                                                                                Mit dem vorliegenden Programm   kann der Tonkopf des Kassetten- geraetes auf eine einzulesende  File eingestellt werden. Fuer   ein MC-Programm ist das MONITOR-Aufzeichnunsverfahren (S) einzu-stellen. KC-kompatible Basicpro-gramme sind mit (P) zu kontro-  lieren. Als Kassettenvorspann   kann ein Pruefton (T) aufge-    zeichnet werden. Die Tonkopfein-stellung auf eine normale File  ist allerdings genauer!                                         Der Tonkopf ist eingestellt,    wenn die im unteren Bildschirm- bereich dargestellten Schwankun-gen am geringsten sind. Die Kom-mandos werden mit Shift4-K      (ASCII=3) abgebrochen.          Das Programm ist vor dem ersten Start (Programmbeginn) frei     relocatibel!
